Cross Distro Developers Camp

2026-05-17

もくもく会を開催しました

ディストリビューション開発もくもく会 2026 年 5 月 を開催しました。 会場は東京の幡ヶ谷にある 株式会社創夢 様の会議室とオンラインのビデオ会議でハイブリッド開催し、参加者は 8 名でした。

参加者は各自で以下の作業を行い、参加者同士で意見交換をしました。

  • mozc が Wayland 環境で選択中・プレエディットの表示を直すパッチを作って upstream に送った (しかし、Ubuntu 26.04 LTS は動いても Ubuntu 24.04 LTS で動かないパッチは受け入れられないと言われて対応を検討中)
  • DebConf27 日本チームに来ていたメールの返信
  • 東京エリア Debian 勉強会の Web サイト更新
  • Debian JP Project の 2026 年度体制アナウンスのメール、web サイト更新
  • Debian JP Project の新ミラーサーバ更改作業の状況確認
  • MainsailOS (3D プリンタ向け OS) の調査、動作確認
  • Chromebook のホスト OS に作った debian コンテナ上で実行する QEMU 環境で RaspberryPi OS (Trixie) を動かせるか試した
  • LibreOffice Ask (QAサイト) の確認
  • 最近リリースされた LibreOffice-26.2.3 のインストールテストと動作確認
  • Ubuntu 26.04 LTS のリリースノート翻訳
  • COSCUP 2026 の CfP レビュー作業
  • かな漢字変換 yuino のサーバ側実装の開発を進めた
  • かな漢字変換 yuino のクライアント側実装の開発を進めた
  • w3m の debian パッケージが古いため、パッケージ内容と upstream を確認していた
  • debian パッケージの作り方、Debian 社会契約を勉強した
  • GNU less を翻訳済、Makefile 修正
  • GNU getent.1 を翻訳済
  • Open Build Service で man-pages-ja を fork して 20260515 版が正しくビルドされることを確認
  • man-pages-ja の RPM をテストするための chroot 環境を作成中 (RPM のインストールでユーザ関連で警告が出て試行錯誤中)

次回開催は、2026/06/14(日) 11:00-18:00 の予定です。
