Cross Distro Developers Camp

2025-11-23

もくもく会を開催しました

ディストリビューション開発もくもく会 2025 年 11 月 を開催しました。 会場は東京の幡ヶ谷にある 株式会社創夢 様の会議室とオンラインのビデオ会議でハイブリッド開催し、参加者は 7 名でした。

参加者は各自で以下の作業を行い、参加者同士で意見交換をしました。

  • LibreOffice Asia Conference 2025 のツアープラン作成
  • Mozc の比較的新しいバージョン (2.32) を openSUSE でビルドを試した
  • JM Project の更新状況確認
  • manpages-ja の Debian パッケージの更新
  • 12 月の Debian 勉強会の案内と発表者募集告知
  • Debian 勉強会 Web サイト のメンテナンス (資料 PDF のリンク追加)
  • Debian JP Project の CD イメージのダウンロードページを更新 (13.2、12.12 向け)
  • Debian Wiki に MiniDebConf Japan 2025 の 報告 を掲載
  • debian-devel メーリングリストのメールを精読
  • Ubuntu 25.10 のリリースノート(日本語版)のレビューと公開
  • Ubuntu 25.10リリース記念オフラインミーティング25.11 向けのスライド資料作成
  • GNU coreutils の翻訳方式と方針を見直し
  • Geeko Magazine のネタ出しや原稿執筆
  • かな漢字変換の Anya のモデルと ibus-anya の修正
  • 連合学習関連の調査
  • Mozilla の製品サポート記事のうち、4 本をレビュー
  • Firefox 146 の翻訳
  • Android 版 Thunderbird の翻訳

次回開催は、2025/12/21(日) 11:00-18:00 の予定です。
